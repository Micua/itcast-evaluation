# Itcast Teaching Management System

## How to use

```bash
$ git clone https://github.com/Micua/itcast-tms.git -b v4.x
$ cd itcast-tms
$ npm install
```

### Run it

```bash
$ npm start
```

### Develop it

```bash
$ npm run serve
$ npm run test
```

or

```bash
$ npm run dev
```
